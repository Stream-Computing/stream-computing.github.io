希姆计算大模型一体机解决方案

方案概述​

在人工智能（以下简称AI）领域，2023年是属于大模型的一年。随着国内百度文心、阿里通义、华为盘古、讯飞星火等一众大模型的登场，如何借助大模型优秀的泛化能力打造更垂直的行业大模型乃至定制企业应用，推理侧正在逐渐成为焦点。

专模专用催生了对私有化商业部署的需求，而其中的关键是对数据安全、算力自主可控以及部署成本的苛刻要求。同时受地缘政治影响，国内半导体产业发展受限，对技术自主可控的需求进一步提升，RISC-V架构由于开源、灵活的特性成为被寄予众望的技术路线。

希姆计算基于RISC-V研发AI计算加速卡STCP920系列发布了大模型一体机（以下简称一体机），为私有化商业部署提供了系统级解决方案，并且正在联合生态伙伴共同打造更多的软硬一体国产化方案。

围绕推理场景系统级交付的目标，我们在研发一体机时着重打造的能力包括：

  • RISC-V创新架构：STCP920系列平衡了高能效、可编程性、通用性，自主可控，并且已经实现大规模量产商用。
  • 灵活部署：可选2U4卡机、4U8卡机，支持部署7B、13B、70B等参数规模的大模型。支持多机互联部署超大规模参数大模型。
  • 高性价比：在单卡算力之上，通过服务器设计支持高强度的多用户并发。
  • 开箱即用：支持主流开源大模型的私有化部署以及本地知识库管理，提供常见的大模型应用开发接口（chat、completion），灵活可扩展。

大模型推理软件栈​

一体机中预装了大模型推理软件栈，架构图如下：

架构图中各模块的作用如下：

  • HPE：希姆计算推出的异构编程环境，提供异构编程接口，同时包含driver、smi、profiler、runtime等使用STCP920所需的组件。
  • stc-dnn：高性能手写算子库，集成了针对STC DSA实现的高性能Kernel。
  • stc_compiler：AI编译器，通过计算图自动调度、算子融合等技术实现模型的自动调优。
  • stc_llm：大模型推理框架，基于算子库和AI编译器，针对DSA特点实现了多种优化技术，models模块支持当前的主流开源大模型，Runtime模块包含显存管理KV cache manager以及Generation pipeline，同时框架提供了兼容Triton的backend，用于支持Triton的调度能力以及continuous batching。
  • k8s-plugin：支持K8s集群中自动申请使用NPU设备，并跟踪NPU设备的健康状况，方便多机扩展。
  • smi-monitor：监控节点的工作状态，提供接口用于查询设备状态。

大模型应用开发者方案​

随着一众大模型的登场，大量开发者也加入了大模型应用的开发大军中，从网页或者手机应用中的各类聊天机器人，到将大模型能力融入现有产品的原型，乃至开发自己的框架进行规模化的应用开发等，大大小小的场景都有发挥大模型能力的尝试。

想要从零开始搭建开发和部署大模型所需的软硬件环境，往往需要很高的成本。希姆计算的大模型一体机致力于解决这个痛点，基于希姆计算的大模型一体机，开发者只需专注于大模型应用本身的开发工作，可以快速部署本地开源/私有模型。

方案架构图如下：

架构图中各模块的作用如下：

  • 开发者通过希姆计算提供的运维组件k8s-plugin和smi-monitor进行服务发现、弹性扩容和状态监控（运维）操作。
  • 开发者按需选择第三方中间件并部署在CPU节点上，例如向量数据库、配置中心、Grafana监控、网路存储，并通过这些中间件提供的标准接口进行相关配置。
  • 开发者通过对接希姆计算提供的应用接口进行开发，并且集成到自己的系统中。通过调用API接口或者集成Python SDK进行大模型应用开发，调用管理接口进行模型配置。

大模型应用终端用户方案​

小微企业或没有充足开发资源的机构，往往需要直接面向终端用户的方案。希姆计算的大模型一体机也提供了开箱即用的方案，除基础的软硬件环境外还会预装必要的中间件，部署聊天机器人应用（持续丰富中），并提供系统组件。终端用户可以直接体验应用效果，并按需进行部署。

方案架构图如下：

架构图中各模块的作用如下：

  • 除基础软硬件外，一体机还预装了应用所需的中间件，例如向量数据库、传统数据库、配置中心、网路存储，以支撑Robot等应用。
  • 系统组件支持应用管理、用户管理、模型管理、节点监控等特性。
  • Robot支持多轮对话、检索增强生成（RAG）等特性。

企业简介​

广州希姆半导体科技有限公司是一家致力于研发以RISC-V指令集架构为基础的人工智能领域专用架构处理器的国内AI芯片公司。自主研发的NeuralScale NPC核心架构是世界领先的、以RISC-V指令集为基础进行扩展、面向神经网络领域的专用计算核心。核心理念是对产品的每一个细节都秉承着精益求精的态度，不断创造更加优秀的产品，迎接各种挑战，实现客观优秀。

广州希姆半导体科技有限公司坚信RISC-V生态必将为AI时代提供澎湃算力，深度参与RISC-V生态的发展，是RISC-V国际基金会最高级别会员、RISC-V董事会成员单位、RISC-V TSC技术委员会成员单位、RISC-V AI方向主席单位和RISC-V Matrix TG成员单位，同时还是中标协RISC-V工作委员会副会长单位、中国智能计算产业联盟副秘书长单位，以及全国信息技术标准化技术委员会-人工智能分技术委员会单位委员和多样性算力产业及标准推进委员会成员单位。
