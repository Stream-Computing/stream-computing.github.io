大模型一体机产品手册

文档概述​

本文档为您介绍希姆计算大模型一体机的产品特点、产品外观、技术规格等产品相关的信息，以及订购相关的信息。

产品简介​

产品外观​

产品特点​

希姆计算大模型一体机以希姆计算国产自研STCP920人工智能加速卡为算力单元，采用自主设计开发的异构软件开发套件，可提供高达2048TOPS的AI算力。希姆计算大模型一体机的产品特点包括：

  • 支持1颗高性能AMD EPYC 7003/7002/7001系列处理器。
  • 支持16个DDR4内存插槽。
  • 最大可提供2048TOPS的AI算力。
  • 卡间互联采用直通设计，提高数据传输效率，降低数据传输时延。
  • 单host设计，以PCIe 4.0实现卡卡之间互联，消除传输瓶颈，提升传输效率。
